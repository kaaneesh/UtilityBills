# UtilityBills

Apartment Utility Bills

This simple single–page app helps caretakers track gas meter readings, export billing CSVs, and send data by email. A new module has been added to allow uploading a photo with a date and description. A single button lets you either take a picture or choose an existing image (the system chooser will offer the appropriate options on phones). When you click Email the compose window opens addressed to **confidentpinnacle.rwa@gmail.com**; you will need to attach the photo manually because mailto links can’t include files. The description text is copied to the clipboard automatically.
