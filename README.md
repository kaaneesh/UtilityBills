# UtilityBills

Apartment Utility Bills

This simple single–page app helps caretakers track gas meter readings, export billing CSVs, and send data by email. A new module has been added to allow uploading a photo – there are separate camera and upload buttons on mobile – with a date and description. When you click Email the compose window opens addressed to **confidentpinnacle.rwa@gmail.com**; you will need to attach the photo manually because mailto links can’t include files. The description text is copied to the clipboard automatically.
